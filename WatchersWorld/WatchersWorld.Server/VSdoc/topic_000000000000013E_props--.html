ProfileInfo Properties

The following tables list the members exposed by the ProfileInfo type.

Properties
 
Name Description
BirthDate Data de nascimento do utilizador. Pode ser utilizada para verificação de idade ou apresentada no perfil do utilizador para fins de personalização.
CoverPhoto URL ou caminho para a foto de capa do utilizador. Esta é uma imagem maior, exibida no topo do perfil do utilizador.
Description Descrição ou biografia breve do utilizador. Pode ser uma declaração pessoal ou qualquer informação que o utilizador deseje partilhar publicamente.
EndBanDate Data de término de um eventual banimento do utilizador, se aplicável.
Followers Número de seguidores do utilizador.
Following Número de utilizadores que este utilizador segue.
Gender Género do utilizador.
ProfilePhoto URL ou caminho para a foto de perfil do utilizador. Esta imagem é normalmente mostrada no perfil do utilizador ou junto ao conteúdo gerado pelo utilizador.
ProfileStatus Estado de visibilidade do perfil, representado por uma enumeração (por exemplo, Público, Privado). Determina a visibilidade ou acessibilidade do perfil a outros utilizadores na plataforma.
StartBanDate Data de início de um eventual banimento do utilizador, se aplicável.
UserId Identificador único do utilizador associado ao perfil, utilizado como chave estrangeira.
UserName Nome de utilizador (username) associado ao perfil do utilizador, utilizado como identificador único e para fins de comunicação. Serve também como chave primária e chave estrangeira referenciando a entidade User.
