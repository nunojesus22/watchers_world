The following tables list the members exposed by the MessagesVisibility type.

Properties
 
Name Description
Id Identificador único para a configuração de visibilidade de uma mensagem.
Message Referência virtual à mensagem associada a esta configuração de visibilidade. Estabelece uma relação de chave estrangeira com a tabela de Messages.
MessageId Identificador da mensagem à qual esta configuração de visibilidade se refere. Este campo é obrigatório.
User Referência virtual ao usuário associado a esta configuração de visibilidade. Estabelece uma relação de chave estrangeira com a tabela de User.
UserId Identificador do usuário para o qual a visibilidade da mensagem é definida. Este campo é obrigatório.
Visibility Um valor booleano que determina se a mensagem é visível para o usuário. Verdadeiro indica que a mensagem é visível, falso indica que não é.
