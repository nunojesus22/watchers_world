MediaController Class

Namespace: WatchersWorld.Server.Controllers
Assembly: WatchersWorld.Server (in WatchersWorld.Server.dll)
Controlador responsável pela gestão de medias no contexto da aplicação, incluindo operações como marcar medias como assistidas ou para assistir mais tarde, verificar se uma media foi assistida ou está marcada para assistir mais tarde, e obter listas de medias assistidas ou para assistir mais tarde de um utilizador específico.
